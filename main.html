Important Notes & Current Status

  • Testing Phase: All tools are currently in an active development and testing phase. We are continuously working to improve performance and add new features based on user feedback.
  • Backend Infrastructure: For the initial development, we are leveraging the free tiers of Firebase and Supabase. This provides us with powerful, scalable, and cost-effective backend infrastructure.
  • Data Security: Data security is a top priority. Despite using free tiers, all data connections are encrypted, and robust access control systems are in place to ensure data integrity and confidentiality.
  • Feedback Welcome: Continuous feedback is crucial for our development process. We encourage all users to report any bugs or suggest improvements to help us enhance these tools.
  • Long-term Goal: Our long-term goal is to develop a comprehensive, in-house ERP system that digitizes and streamlines all of QGO Cargo's operational needs, significantly boosting efficiency and reducing manual workload.

This project is a collaborative effort, made possible by the invaluable support from the QGO Cargo team.

Purpose

To digitize and manage all shipment/job files in one place.

Key Features

  • Secure login with roles (Admin, Checker, Staff).
  • Create job files with shipper, consignee, invoice, and freight details.
  • Built-in charges table for billing (freight, handling, clearance, storage).
  • Approval workflow from checker to admin.
  • Analytics dashboard: monthly/yearly profit, salesman performance.
  • Print/Export A4 receipts and PDF outputs.

Value & Benefits

  • Eliminates paper files, leading to fewer errors and faster clearance.
  • Saves cost by reducing the need for carbon copies and printing.
  • Data is secure, backed up, and easy to retrieve.

Purpose

To track deliveries and collect customer proof in real-time.

Key Features

  • Drivers can log in, get assignments, and update delivery status.
  • Capture digital signatures from customers as proof.
  • Auto-generate QR code receipts for deliveries.
  • Public links for customers to track and confirm deliveries without a login.
  • Feedback system for customer ratings and comments after delivery.
  • Dashboard with Pending, Completed, and Total deliveries statistics.

Value & Benefits

  • Provides real-time tracking of every delivery.
  • Allows for faster dispute resolution with digital proof.
  • Builds customer trust through transparency and a feedback mechanism.

Purpose

To manage and monetize warehouse storage for third-party client cargo.

Key Features

  • Client management portal to add and track customer details.
  • Cargo intake form to log shipment details (client, weight, dimensions).
  • Automated billing system with configurable daily/weekly rates.
  • Set custom free storage days per client or shipment.
  • Cargo release function that stops billing and generates a final invoice.
  • Dashboard for analytics on storage utilization and revenue.

Value & Benefits

  • Turns warehouse space into a reliable revenue stream.
  • Eliminates manual billing errors, preventing revenue loss.
  • Provides clients with transparent and accurate invoices, building trust.
  • Streamlines the entire process from cargo intake to release.

Purpose

To manage cargo storage and inventory in a private company warehouse.

Key Features

  • Shipment intake form with barcode scanning capability.
  • Detailed fields: barcode, shipper, consignee, weight, pieces, rack location.
  • Auto-calculates internal storage costs and handling fees.
  • Settings page to configure rates and charges.
  • Release option that generates internal records and invoices (PDF).
  • Full Admin Panel with activity logs for auditing.
  • Analytics: storage usage and billing trends via charts.

Value & Benefits

  • Ensures accurate and transparent billing, preventing revenue loss.
  • Enables quick search and retrieval with barcode scanning.
  • Invoices and records are generated instantly.
  • Provides complete warehouse visibility for management.

How it Works

A one-click solution that downloads all important job file data into an Excel format for local backup.

Benefits & Security

Reduces risk of data loss and eliminates manual data entry for reports. Critical data backup is available in seconds. As a client-side tool, your data remains securely on your system.

Future Plans

Plans include automated cloud backups (Google Drive) and date-range selection for targeted downloads.

Purpose

Designed for accountants to generate and download comprehensive reports for financial analysis and auditing.

Key Features

Allows one-click download of all job files into a single Excel file. The data can be easily filtered by salesman, the user who created the file, or by consignee, making it highly flexible for accounting needs.
